— Common questions

Things people ask first.

Do I own the code LingCode generates?
Yes — entirely. LingCode emits a real Xcode project to your machine or your GitHub. We don't hold a license, we don't watermark anything, and you can leave with the project at any time.
What's LingModel and why is it metered?
LingModel is the managed inference plan baked into Pro and Max Pro — you don't bring a key, we route requests for you. Pro removes the Free-tier prompt cap; Max Pro lifts the per-day and per-month token throughput so heavy users don't hit a wall.
Can I bring my own model API key?
Yes — on every tier, including Free. Plug in Claude, DeepSeek, OpenAI, Gemini, Mistral, or any OpenAI-compatible endpoint and it stays unlimited. LingModel is for people who'd rather not manage keys.
What's Deep Agent?
An agent loop that runs server-side via the Anthropic Agent SDK with longer task horizons than the local loop. Lives in the browser playground. Available on Pro and Max Pro — Max Pro gets a higher token budget per task.
Do I need an Apple Developer account?
Not to build apps with LingCode. You'll need one (free for sideloading, $99/yr for distribution) when you want to push to TestFlight or the App Store. We'll walk you through it.
Can I switch plans or cancel?
Switch any time, prorated. Cancel any time, no questions. Your projects stay accessible; the Xcode export stays yours forever.
Is there a team plan?
Not yet — every plan is per-seat for now. If you want centralized billing for a team, drop us a line and we'll sort something out.
How does this compare to Lovable, Bolt, Cursor?
Short version: they output React. We output native Swift, Kotlin, and React. Long version is on the homepage compare table →

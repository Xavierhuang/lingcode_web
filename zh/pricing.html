— 常见问题

大家最先问的。

我拥有 LingCode 生成的代码吗？
拥有——完全拥有。LingCode 会把一个真实的 Xcode 项目输出到你的电脑或 GitHub。我们不持有任何许可、不加任何水印，你随时可以带着项目离开。
LingModel 是什么，为什么要计量？
LingModel 是内置于 Pro 与 Max Pro 的托管推理套餐——你无需自带密钥，由我们替你路由请求。Pro 取消了免费档的提示次数上限；Max Pro 进一步提升每日与每月的 token 吞吐量，让重度用户不会撞墙。
我可以自带模型 API 密钥吗？
可以——所有档位都支持，包括 Free。接入 Claude、DeepSeek、OpenAI、Gemini、Mistral 或任意 OpenAI 兼容端点，且保持无限量使用。LingModel 是为那些不想自己管理密钥的人准备的。
Deep Agent 是什么？
一种通过 Anthropic Agent SDK 在服务端运行的 Agent 循环，任务跨度比本地循环更长。它存在于浏览器演练场中。Pro 与 Max Pro 可用——Max Pro 每个任务的 token 预算更高。
我需要 Apple 开发者账号吗？
用 LingCode 构建应用不需要。当你想推送到 TestFlight 或 App Store 时才需要一个（侧载免费，发布每年 $99）。我们会一步步带你完成。
我可以切换套餐或取消吗？
随时切换，按比例计费。随时取消，无需说明理由。你的项目始终可访问；Xcode 导出永远归你所有。
有团队套餐吗？
暂时没有——目前每个套餐都按席位计费。如果你想为团队做集中计费，给我们留言，我们会帮你想办法。
这与 Lovable、Bolt、Cursor 相比如何？
简短版本：它们输出 React。我们输出原生 Swift、Kotlin 和 React。详细版本见首页对比表 →

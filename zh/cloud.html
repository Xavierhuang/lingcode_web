箱子里有什么

一个真正的应用在后端所需的一切。

数据

Postgres 数据库

真正的 SQL 数据库——创建表、运行查询、管理迁移。不是玩具式的键值存储。

认证

用户与登录

邮箱密码，外加 Google 和 GitHub OAuth，可选 MFA——按后端隔离。

存储

文件存储

公开与私有存储桶，背靠带 CDN 的对象存储，大文件可预签名直传。

实时

实时更新

订阅行变更，数据变化时主动推送——无需轮询。

函数

无服务器函数

在沙箱运行时里运行你自己的代码，支持密钥注入与按档超时。

搜索

向量搜索

用 pgvector 与托管嵌入对数据做语义搜索——为 AI 功能而生。

Agent 原生

AI 可直接使用

后端通过 MCP 把自己暴露给 Agent，AI 可描述、查询并存储文件——无需胶水代码。

邮件

事务邮件

从后端发送注册、重置和通知邮件，按档每日额度。

工作原理

连接一次，其余交给构建。

01

连接

在 Mac 应用或 /try 里连接一个后端。我们会配置 Postgres、认证和存储，并写好 .mcp.json——即时上线，自带 HTTPS。

02

构建

你的应用通过 SDK 访问它，AI Agent 通过 MCP 读写它。建表、加认证、存文件——手动或交给 AI。

03

扩展

文件存在弹性对象存储上，数据库随套餐扩展。每档都有清晰的额度，触顶前会提醒你。

信任基石

项目经得起 AI 折腾。

AI 可能悄悄搞坏一个 Xcode 工程、写乱 AndroidManifest.xml、或者把 Gradle 配置拍扁。LingCode 假定它一定会出错，于是每次改动都在你注意到之前就能回退。

编辑前快照

Agent 落笔之前，project.pbxproj、Info.plist、build.gradle、AndroidManifest.xml 与 *.entitlements 的原始副本会先写入 ~/Library/Application Support/LingCode/WorkProtection/。恢复面板一键还原，和 Git 无关——即便没提交也能救回来。

按 hunk 接受 / 还原

每一次 AI 编辑都会在文件上显示带语法高亮的 diff 卡片。每个 hunk 独立接受或还原——不再是"全部通过"或"全部丢弃"。审阅 40 个文件的重构也不用把控制权交出去。

语义化时间回溯撤销

专为 AI 操作维护的多文件撤销栈，与编辑器原生撤销互相独立。一步回退跨文件重构——即使你之后继续打字也不会丢失你自己的编辑。

四种聊天模式

60+ 原生 Swift 工具：构建、运行、调试、读取代码库、直接编辑文件。四种模式：Agent（自主执行、逐步审批）、Plan（先设计再编辑）、Ask（问答、不修改）、Debug（专注排错）。3 级 Token 递增、自动上下文压缩、跨会话结构化记忆。

Worktree 隔离

把 Agent 送进一个独立的 git worktree 做高风险实验。做完之后合并、保留分支或丢弃——主分支毫发无损。完整信任模型见 上方章节。

Android 开发

Gradle 构建、AVD 模拟器、adb 安装、logcat 实时流、Kotlin/JDWP 调试器、签名 AAB 输出——同一个 Mac 应用，无需 Android Studio。

规则 (.mdc)

支持 WORKSPACE.md、.cursorrules、.cursor/rules——直接导入你的 Cursor 规则，无需迁移。

Magic 工作流

Install — 检测 npm、pip、Cargo、CocoaPods 等。Git — 暂存、AI 提交信息、推送。Deploy — 引导部署到常见托管平台。

技能

斜杠命令和工作流脚本：/review、/commit、/security、/mode、/help，也支持自定义。Cursor 兼容——LingCode 直接读取 .cursor/skills、.agents/skills、~/.cursor/skills 中的 SKILL.md 文件。输入 / 即可开始。

扩展

VS Code 风格的扩展商店。LSP（Swift、Python、TypeScript、Go、Rust、C++）、插件（Prettier、ESLint、Next.js、Vue）和工具。一键安装。

多 Claude Code 会话

并行 Claude Code 标签页——每个会话独立运行。

LSP 智能感知 + FIM ghost 补全

真正的 language server（SourceKit、Kotlin、TS、Python、Go、Rust）：前缀补全、诊断、跳转到定义。加上 Copilot 风格的 AI ghost 文本，使用 FIM（中间填充）——补全同时理解你已经敲下的和光标后面的内容，不只看前缀。可接云端或本地模型。Tab 接受，Escape 取消。

HTTP 代理检查器

localhost 正向代理，用于调试你 App 的明文 HTTP 流量——无需安装根证书。实时捕获方法、URL、请求头、请求体和响应状态。Agent 在排查抖动的请求时可读取最近的事务。

浅色与深色主题

浅色模式覆盖每一处——编辑器、终端、AI 聊天、diff、审批对话框。终端配有专属浅色 ANSI 调色板。跟随系统外观或手动切换。

会话持久化

AI 标签页及其对话在退出和重启间保留。Cmd+Q 时同步落盘，重新打开自动恢复。从你离开的地方继续。

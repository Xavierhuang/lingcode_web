配置 API 密钥

将 Claude、DeepSeek、GPT-4、Gemini 或任意 OpenAI 兼容服务商接入 LingCode，释放它们在操场中的全部能力。

你将学到什么

  • 如何获取各服务商的 API 密钥
  • 如何在 LingCode 中安全存储密钥
  • 如何测试连接是否正常
  • 如何管理多个服务商
  • 如何使用自定义端点（LingModel、本地模型等）

概述：API 密钥的工作原理

LingCode 通过 API 密钥连接 AI 服务商——这是一个证明你拥有账户并有权使用该服务的私密令牌。你的密钥永远不会离开你的浏览器；它们只会直接发送给服务商的 API，而不会经过 LingCode 的服务器。

安全提示：API 密钥属于敏感信息。请勿在公开场合分享、提交到 git，或粘贴到不可信的网站。LingCode 会将密钥安全地存储在你的浏览器本地存储中。

配置 Claude（Anthropic）

1

创建 Anthropic 账户

访问 console.anthropic.com，注册或登录。

2

生成 API 密钥

进入 API 密钥页面，点击"Create Key"，为密钥取个名称（例如"LingCode"），然后复制完整的密钥。

3

粘贴到 LingCode

在操场中，点击底部的"Advanced"按钮，找到 Claude 一行，粘贴你的密钥，然后点击"Save"。

配置 DeepSeek

1

创建 DeepSeek 账户

访问 platform.deepseek.com，注册或登录。

2

创建 API 密钥

在 API 密钥页面，点击"Create API Key"，复制并妥善保存。

3

添加到 LingCode

在操场中，点击"Advanced"，找到 DeepSeek，粘贴密钥并保存。

费用提示：DeepSeek 的价格远低于 Claude 或 GPT-4——通常便宜 10 倍以上。非常适合批量测试和实验。

配置 OpenAI（GPT-4、GPT-4o）

1

创建 OpenAI 账户

访问 platform.openai.com，注册或登录。如果尚未添加支付方式，请先完成添加。

2

生成 API 密钥

在左侧栏进入"API keys"，点击"Create new secret key"，然后复制密钥。

3

在 LingCode 中配置

在操场中，点击"Advanced"，找到 OpenAI，粘贴密钥并保存。

测试连接

添加密钥后：

  1. 进入操场
  2. 点击服务商按钮（Claude、DeepSeek 等）
  3. 输入一个简单的提示词："Hello, can you hear me?"
  4. 点击"Send"
  5. 如果服务商有响应，说明连接成功 ✓

同时使用多个服务商

LingCode 的强大之处在于可以同时向多个服务商发送同一个提示词，并对比结果。操作方法如下：

  1. 为多个服务商添加 API 密钥（Claude、DeepSeek、GPT-4 等）
  2. 在操场中，点击选中多个服务商按钮
  3. 输入提示词并点击"Send"
  4. 所有服务商将并行生成回答——挑选最佳结果即可
使用技巧：在重要构建中善用这一功能。同时在 Claude、DeepSeek 和 GPT-4 上运行你的提示词，对比质量与成本，选出最优方案。

删除或轮换密钥

从 LingCode 中删除密钥：

  1. 在操场中点击"Advanced"
  2. 找到对应的服务商行
  3. 清空密钥字段并保存

如需轮换 API 密钥（生成新密钥并吊销旧密钥），请在服务商的控制台中操作，然后在 LingCode 中更新为新密钥。

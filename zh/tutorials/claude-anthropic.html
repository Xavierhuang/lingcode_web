如何在 LingCode 中使用 Claude (Anthropic)？

TL;DR：在 console.anthropic.com 获取 API 密钥，粘贴到 设置 → AI 服务商 → Claude，然后选择一个 Claude 模型（Opus 4.7 适合最复杂的任务，Sonnet 4.6 是默认推荐，Haiku 4.5 速度最快）。工具调用、MCP 和流式输出均开箱即用。

将 Anthropic 最强大的 AI 模型 Claude 接入 LingCode。Claude 擅长细腻的设计任务和复杂推理。

为什么选择 Claude？

Claude 是 Anthropic 的旗舰 AI 模型，以下几点尤为突出：

  • 设计理解力：Claude 擅长理解设计系统、品牌规范和视觉层级，非常适合打磨精致的 UI/UX 设计。
  • 复杂推理：能够处理多步骤问题、系统架构设计和战略思考。
  • 长上下文：可处理非常大型的文档和代码文件，而不会丢失上下文。
  • 安全与可靠：Anthropic 专注于减少幻觉和偏差。
  • 工具调用：Claude 可使用工具和 MCP 服务器，支持高级工作流。
最适合：高质量路演 PPT、精致仪表盘、品牌感设计，以及需要深度推理的创意项目。

入门

1

注册 Anthropic 账号

前往 console.anthropic.com 创建账号。你需要提供：

  • 电子邮箱
  • 密码
  • 账单信息（支付方式）
2

创建你的第一个 API 密钥

登录后，在左侧边栏找到 "API Keys"，点击蓝色的 "Create Key" 按钮。你可以选填一个名称（例如 "LingCode"）以便识别。

你的密钥格式如下：sk-ant-v0-XXXXXXXXXXXX...

请妥善保管此密钥——像对待密码一样保护它。切勿提交到 git 或公开分享。

将 Claude 添加到 LingCode

3

打开演练场并进行配置

前往 /try.html，滚动到页面底部，点击 "Advanced" 按钮展开服务商配置。

4

粘贴你的 API 密钥

在服务商列表中找到 "Claude" 一行，点击输入框并粘贴你的 Anthropic API 密钥（sk-ant-v0-...）。

点击 "Save"，密钥将安全存储在你的浏览器本地（不会发送到 LingCode 服务器）。

测试连接

5

运行一个测试提示词

在文本区域输入一个简单的提示词：

Build a simple landing page for a coffee shop with a hero, menu preview, and contact form.

点击 "Claude" 服务商按钮将其选中（按钮应高亮），然后点击 "Send"。如果 Claude 在预览区返回了完整的 HTML 页面，说明连接成功！✓

可用的 Claude 模型

Anthropic 持续更新 Claude 的能力。LingCode 中可用的模型包括：

  • Claude 4：能力最强的模型，适合复杂的设计和推理任务
  • Claude 3.5 Sonnet：速度快且聪明，兼顾速度与质量
  • Claude 3 Opus：上一代旗舰模型，依然非常出色
  • Claude 3 Haiku：快速轻量，适合简单任务

LingCode 会自动为你的账号选择最佳可用模型。你可以在 Anthropic 控制台查看配额和用量。

定价与用量

Claude 按 token 计费（输入和输出分别计算）。截至 2026 年：

  • 一次典型的路演 PPT 提示词约花费 $0.10–0.30
  • 一个仪表盘或复杂页面约花费 $0.20–0.50
  • 简单的编辑或续写约花费 $0.01–0.10

在 console.anthropic.com/account/usage 查看用量。你可以设置消费上限，避免意外超支。

省钱建议：学习和实验阶段使用 LingModel（免费），在需要最高质量的重要项目时（路演 PPT、投资人演示、关键设计）再切换到 Claude。

常见问题排查

Q：提示"Invalid API key"错误

A：请确认你已从 Anthropic 控制台复制了完整的密钥（应以 sk-ant-v0- 开头），注意避免首尾多余的空格。

Q：Claude 响应很慢 / 配额已耗尽

A：请在 console.anthropic.com/account/limits 查看用量面板。如果触发了速率限制，等待几分钟即可恢复；如果 token 预算已用完，请向账单账户充值。

Q：应该选择哪个 Claude 模型？

A：LingCode 会自动为你选择最新可用的 Claude 模型，无需手动配置。

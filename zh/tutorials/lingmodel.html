LingModel 是什么，如何使用？

简而言之：LingModel 是 LingCode 内置的托管推理服务，无需 API 密钥。Starter 套餐免费使用（有额度限制），Pro 和 Max Pro 套餐不限量。在服务商菜单中选择 LingModel，直接输入提示词即可开始，所有请求均通过 LingCode 的托管代理处理。

最简单的上手方式。无需 API 密钥，无需配置，免费使用。LingModel 是内置于 LingCode 的托管 AI 服务。

LingModel 是什么？

LingModel 是由 LingCode 托管的内置 AI 服务商。它驱动所有快速入门示例和"Try:"模板，也是你第一次打开操场时的默认选项。你不需要：

  • API 密钥
  • 单独的账户
  • 支付方式（免费套餐）
  • 任何配置

打开操场，直接开始构建。

免费套餐与付费套餐

免费套餐：100 次免费查询，一次性发放（非每月重置）。非常适合学习、探索和日常使用。

付费套餐：无限次查询。按需付费或订阅以获取更高额度。登录后可查看配额和升级选项。

入门建议：用免费套餐学习 LingCode、构建几个项目。当你需要无限访问或优先支持时，登录并升级即可。

使用 LingModel

1

打开操场

前往 /try.html。LingModel 已作为默认服务商预先选中。

2

输入提示词

点击文本框，描述你想构建的内容：

  • "为一家 AI 初创公司构建一个落地页"
  • "创建一个数据分析仪表盘"
  • "生成一份商业提案模板"
3

点击"发送"并等待

LingModel 将实时生成你的设计。你会看到 HTML 在预览面板中随着流式输出逐步渲染。

4

下载你的作品

完成后，点击三点菜单并选择：

  • 下载 .html：保存为文件
  • 导出 PDF：用于打印或分享
  • 导出 PPT（如果你的设计包含幻灯片）

免费套餐限额

免费套餐共提供 100 次免费查询——这是一次性额度，用于体验 LingModel，不会每月重置。"一次查询"指一条提示词加上后续所有"继续"请求。

用量示例：

  • 1 个路演幻灯片提示词 = 1 次查询
  • 1 个路演幻灯片 + 3 次修改（"调暗颜色"、"添加定价页"等）= 4 次查询
  • 10 个不同的小设计 = 10 次查询

快用完了？快到限额时你会看到提示。你可以：

  • 登录并升级到付费套餐，享受无限访问
  • 如果已配置 API 密钥，切换到其他服务商（Claude、DeepSeek、GPT-4）
提示：对于学习而言，100 次免费查询通常足够评估 LingModel 并构建几个项目。开始正式开发时再切换到付费套餐。

何时升级到付费套餐

在以下情况下，建议升级你的 LingModel 账户：

  • 免费查询已用完，但你还想继续构建
  • 你希望无限构建，不再担心配额
  • 你需要优先支持或更快的处理速度
  • 你想在 LingCode Mac 版中使用 LingModel（即将推出）

升级步骤：

  1. 点击操场右上角的 "登录" 按钮
  2. 创建 LingCode 账户或直接登录
  3. 进入账户设置，选择一个套餐
  4. 添加支付方式

改用其他服务商

如果你不想升级 LingModel 但又需要无限构建，可以使用其他服务商：

  • Claude（Anthropic）：配置 Claude 的 API 密钥，直接按量付费给 Anthropic
  • DeepSeek：比 Claude 更经济，适合快速构建
  • GPT-4：质量与速度均衡

参阅 Claude、DeepSeek 和 OpenAI 的教程，开始使用这些服务商。

LingModel 与其他选项的对比

常见问题

问：使用 LingModel 需要账户吗？

答：免费套餐无需账户。如果想升级到无限查询或查看用量，登录即可。

问：LingModel 在 Mac 版中可用吗？

答：目前 LingModel 在 Web 操场中可用。Mac 版即将支持 LingModel，敬请期待。

问：免费查询用完了怎么办？

答：你会看到配额已用尽的提示。登录后升级到无限套餐，或使用自己的 API 密钥切换到其他服务商。

问：可以同时使用 LingModel 和 Claude 吗？

答：可以。配置好 Claude 的 API 密钥后，在操场中同时选择 LingModel 和 Claude，即可用相同的提示词对比两者的输出结果。

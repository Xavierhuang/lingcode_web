从 Hover 购买域名

Hover 将自己定位为专注于域名注册的服务商——只卖域名，别无其他。结账流程没有任何强推，客服专业可靠，界面干净简洁。价格比廉价注册商略高，换来的是更顺畅的体验。

在"不废话"域名注册商这个细分市场里，有一小部分甚至不追求最低价，Hover 就是其中最典型的代表。他们的定位很清晰：只卖域名和邮箱，不夹带任何附加推销。结账体验比 GoDaddy 清爽得多，也比 Porkbun 更精致一些，价格则比廉价注册商贵出几美元。

适合 Hover 的用户，是那些愿意多花一点钱来换取省时省力的人——不用费心屏蔽各种交叉推销，比每年省下 3 美元更值得。对于一两个个人域名，这笔账算得过来；但如果你管理几十个域名，价差就不容小觑了，这时候 Cloudflare 或 Porkbun 会更划算。

Hover 隶属于 Tucows 旗下，Tucows 同时运营 OpenSRS，是全球规模最大的后端域名注册服务商之一。所以尽管 Hover 品牌不大，底层基础设施却是工业级的——稳定的企业背书和靠谱的客服，都是你多付那点钱买到的东西。

你将学到什么

  • Hover 的定价结构，以及为何高于廉价注册商
  • 极简结账流程——不需要取消勾选任何选项
  • 邮件转发与真实邮箱的区别——Hover 卖的是哪种
  • DNS 管理面板——直观无惊喜
  • 哪些情况不适合用 Hover（应该选更便宜的方案）

第一步：搜索并比较价格

1

每年同价，和优质注册商一样透明

在 hover.com 上搜索域名，显示的价格就是续费价格——没有首年优惠，没有星号注释。.com 大约在 15–17 美元之间，比 Cloudflare 贵约 5 美元，比 Namecheap 续费年贵约 2 美元。其他常见顶级域名也遵循同样的规律：略贵，但定价透明。

Hover 支持多种顶级域名，包括许多国家代码域名（.co.uk、.ca、.de 等）。对于特殊顶级域名，Hover 未必是最便宜的选择，但他们能干净利落地处理各地区的注册要求。

第二步：加入购物车，结账，搞定

2

购物车里只有一个项目

购物车里显示的就是你的域名。WHOIS 隐私保护免费附送；一个地址的邮件转发（仅接收）也免费包含。没有 SSL 推销，没有托管推销，没有建站工具推销。如果你需要真实邮箱（收发邮件），可以单独购买，但结账时不会主动推给你。

填写联系信息、支付方式，确认即可。整个流程比读完这段话还快。

Hover 邮箱 vs. 邮件转发：免费的邮件转发会将收到的邮件转发到你现有的邮箱。Hover 也提供真实邮箱服务（可收发、支持 Webmail 和 IMAP），每个邮箱约 20 美元/年。如果你需要能发送邮件的 hello@yourdomain.com，这是一个选择——不过 Fastmail 或 Google Workspace 通常性价比更高。

第三步：DNS 管理面板

3

简单直接，没有意外

Hover 的 DNS 界面是标准的记录列表，支持内联编辑——A、AAAA、CNAME、MX、TXT、SRV 一应俱全。没有花哨设计，也不会碍手碍脚。添加记录，保存，完成。

关于具体配置——哪些记录类型对应哪些目标、解析传播行为、验证方式——请参阅将域名连接到服务器。DNS 操作本身在每家注册商都是一样的，唯一的区别只是点击的界面不同。

第四步：客户支持——真正的差异化优势

4

真人支持，北美工作时间

Hover 真正的卖点——也是廉价注册商普遍缺乏的——是专业的电话和邮件支持。如果域名转移卡住了，如果某个顶级域名的注册要求不够清晰，如果需要找回账户，你可以直接联系到懂行的工作人员。等待时间短，客服有权限直接解决问题。

大多数开发者用不到这项服务；但少数人——通常是域名对业务至关重要、出了问题的时候——会觉得这点价差完全值得。

第五步：何时应该选别家

5

对价格敏感的场景

  • 域名数量多。在 Hover 注册 5 个域名，比在 Cloudflare/Porkbun 每年多花 25–50 美元。积累数十年，差距相当可观。规模大了，就该优先考虑价格。
  • 临时或实验性域名。如果不确定第一年后还会不会续费，为用不上的客服付溢价并不划算。选最便宜的注册商就好。
  • 需要 API 自动化。Hover 没有公开 API。如果你需要以编程方式管理域名，这是一票否决——Porkbun、Cloudflare、Gandi、AWS 都有 API，Hover 没有。

第六步：购买后的后续操作

6

标准后续步骤

  • 将 DNS 指向你的网站所在位置——参见DNS 配置。
  • 在账户设置中开启两步验证（2FA）。
  • 如果使用Magic Deploy，按照托管方的 DNS 说明操作——通常只需添加一两条记录。
无编程访问接口：Hover 不提供公开 API。如果你使用的工具依赖注册商端的自动化（Terraform 管理 DNS、脚本自动更新记录），Hover 无法满足需求。在将自动化程度高的域名迁移进来之前，请提前规划好。

如何在 LingCode 中使用 OpenAI（GPT-4o）？

TL;DR：在 platform.openai.com 获取 API 密钥，粘贴到 设置 → AI 提供商 → OpenAI，然后选择模型（GPT-4o 适合常规任务，GPT-4o-mini 成本更低，o1 适合复杂推理）。支持原生函数调用工具使用和 MCP。

将 GPT-4o 或其他 OpenAI 模型接入 LingCode，在生成质量、速度与成本之间取得最佳平衡。

为什么选择 OpenAI？

OpenAI 的模型（尤其是 GPT-4o）具备以下优势：

  • 均衡的质量：在设计任务上表现出色，价格却比 Claude 更实惠
  • 高可靠性：经数百万开发者验证的成熟模型
  • 强通用性：擅长 HTML、CSS 及创意设计
  • 较快速度：比 Claude 快，比 Claude 便宜，但比 DeepSeek 稍慢
最适合：追求合理价格下扎实效果的场景。介于 DeepSeek（经济实惠）和 Claude（高端精品）之间的中间选择。

入门

1

创建 OpenAI 账号

访问 platform.openai.com 注册或登录。你需要准备：

  • 电子邮箱地址
  • 密码
  • 支付方式（信用卡）

在生成首个 API 密钥之前，请先完成账单信息的填写。

2

生成 API 密钥

登录后，在侧边栏点击 "API Keys"，再点击 "Create new secret key"，完整复制生成的密钥。密钥格式如下：sk-proj-...

请妥善保管此密钥。切勿分享给他人或提交到 git 仓库。

将 OpenAI 添加到 LingCode

3

打开 playground 并进行配置

前往 /try.html，向下滚动到页面底部，点击 "Advanced" 展开提供商配置选项。

4

粘贴 API 密钥

找到 "OpenAI" 一行，点击输入框并粘贴你的密钥（sk-proj-...）。

点击 "Save"，密钥将安全地保存在你的浏览器中。

测试连接

5

发送测试提示词

输入一个简单的提示词：

Build a simple landing page with a hero section, features list, and CTA button.

点击 "OpenAI" 提供商按钮选中它，再点击 "Send"。如果 OpenAI 返回完整的 HTML 页面，说明连接成功！✓

OpenAI 模型说明

OpenAI 提供多种可用于设计任务的模型：

  • GPT-4o：最新、最快、能力最强，适合大多数设计任务，推荐使用。
  • GPT-4 Turbo：较旧版本，速度略慢，整体表现依然优秀。
  • GPT-3.5 Turbo：更旧的版本，速度更快、价格更低，但在设计任务上效果稍逊。

LingCode 默认自动选用 GPT-4o（最新版本），无需手动指定模型。

OpenAI 定价参考

截至 2026 年，各类任务的大致费用：

  • 落地页：约 $0.10–0.20
  • 演示文稿：约 $0.15–0.30
  • 复杂仪表板：约 $0.20–0.40

比 DeepSeek 贵，但在同等质量下比 Claude 便宜。

适用场景

  • 均衡需求：希望以适中的成本获得良好效果
  • 客户交付：质量足够可靠，且比 Claude 更经济
  • 实验探索：比 Claude 便宜，同时比 DeepSeek 更稳定

智能工作流：DeepSeek + GPT-4 + Claude

面对重要任务时，可以策略性地组合使用这三个提供商：

  1. 先用 DeepSeek 快速生成低成本原型
  2. 效果不错后，切换到 GPT-4 进一步打磨，成本适中
  3. 仍未满意时，再用 Claude 追求最佳品质，成本较高

这套工作流通常能在质量与成本之间找到最理想的平衡点。

常见问题排查

问："Invalid API key"（API 密钥无效）错误

答：请确认已从 OpenAI 控制台完整复制密钥（以 sk-proj- 开头），并注意检查是否含有多余空格。

问：OpenAI 返回速率限制错误

答：请前往 platform.openai.com/account/limits 查看你的账号使用情况。可能已达到用量上限，稍等片刻后重试，或向账号充值以提高额度。

问：我正在使用哪个 GPT-4 模型？

答：LingCode 默认自动使用 GPT-4o（最新且速度最快的版本），无需手动选择。

问：能否同时使用 GPT-4 和 Claude？

答：可以。分别配置两个提供商后，在 playground 中同时选中两者并发送提示词，即可并排对比两者的输出结果。

API 用量监控

要追踪你的费用消耗：

  1. 登录 OpenAI 控制台
  2. 进入 "Usage" 页面查看 API 费用
  3. 进入 "Limits" 页面设置消费上限并查看速率限制信息

建议设置每月消费上限，避免意外超支。

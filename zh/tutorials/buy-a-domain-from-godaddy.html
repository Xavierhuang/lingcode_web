从 GoDaddy 购买域名

GoDaddy 是人人耳熟能详的注册商，也是最值得仔细看清楚条款的那一个。他们的域名本身没问题，但结账流程经过精心设计，目的是让你多花钱，而且续费价格每个周期都在涨。教你如何干净利落地完成购买。

GoDaddy 是全球最大的域名注册商，遥遥领先。他们靠激进的营销策略做大，是大多数非技术用户接触到的第一家注册商，并且在每个结账环节都会推销周边服务。如果你在读这篇教程，大概是因为有人告诉你"去 GoDaddy 买个域名就行"，或者你曾看过他们的广告，再或者你已经有域名在那边、想知道怎么应对。

"别买附加服务"这条建议在每家注册商都适用，但在 GoDaddy 尤为关键——同一个域名同一年，付 $12 和付 $80 的差距就在于此。默认购物车里会塞进 Domain Privacy（收费，而其他注册商是免费赠送的）、SSL 证书（根本不需要从注册商这里买——现代主机全都免费提供）、虚拟主机、Microsoft 365 邮箱，以及至少三项没有明显勾选标记的其他服务。

这篇教程讲的是最简路径。你可能有正当理由留在 GoDaddy——也许是继承了别人的域名，也许品牌知名度对客户交付很重要，也许你最初就在这里注册、不想迁移。这些都无可厚非。重点是：以相当于便宜注册商的价格，只买一个域名，结束交易。

你将学到

  • GoDaddy 会试图塞进购物车的五种附加服务
  • 为什么 GoDaddy 的续费价格是业内最高的
  • 结账时的"取消勾选"流程
  • Domain Privacy：在 GoDaddy 要收钱——但还是值得开启
  • 购买后何时考虑将域名迁走

第一步：搜索域名

1

显示的价格具有误导性

godaddy.com 的搜索框会显示一个价格——通常是"第一年仅 $0.99！"之类——这只是第一年的促销价。从第二年起每年续费，.com 域名通常是 $18–25。这才是真正重要的数字，请按此预算。

如果你想要的域名显示为"高级域名"，说明它已被人抢注——有人在批发市场上花钱持有这个名字，注册局因此收取高得多的价格。GoDaddy 的高级域名可以高达数百乃至数千美元，几乎从不值得购买；换一个名字吧。

第二步：购物车——取消勾选所有附加项

2

五个条目，只有一个值得要

将域名加入购物车后，GoDaddy 会自动添加（或强烈推荐）几项服务，逐一处理：

  • Domain Privacy（域名隐私）。$10–20/年。保留。不开隐私保护，你的真实姓名和地址会在 WHOIS 上公开。收费确实烦人（其他注册商免费提供），但保护价值高于费用。
  • SSL 证书。$70+/年。坚决跳过。Let's Encrypt 和所有现代主机都提供免费 SSL。付费 SSL 证书只在有特定合规需求时才有意义，普通网站纯属浪费。
  • 虚拟主机。$5–10/月。跳过。如果你有 LingCode，就有 Magic Deploy。从注册商购买虚拟主机通常不划算。
  • Microsoft 365 邮箱。$5–10/月。除非你真的需要这个域名的邮箱，否则跳过。如果确实需要，Fastmail 或 Google Workspace 通常性价比更高。
  • 网站建设工具。$10–25/月。跳过。与你无关。

取消勾选后四项后，购物车里应该只剩：域名 + Domain Privacy。就这些。

第三步：续费年限的账

3

GoDaddy 的续费价格是行业最高

第一年是 $0.99、$9.99 或某个促销价，从第二年起，.com 续费预期 $18–25。如果你管理一批域名，五年下来累积下来会多付 $100 以上——而在 Cloudflare 或 Porkbun 同样的域名要便宜得多。

如果你对价格敏感，这是迁出最有力的理由（见第七步）。如果不在意，那就明白你在为什么买单：GoDaddy 的价格包含了他们的客服团队和庞大的营销预算——这是那些低价注册商没有的成本。

第四步：自动续费——保持开启

4

意外失去域名的代价远大于续费成本

自动续费会在到期前几周从你的银行卡扣款。和所有注册商一样的建议：保持开启。因到期而失去域名，损失远比多付一年域名费严重得多。

第五步：通过"我的产品"页面管理域名

5

DNS 面板藏得深，但功能可用

购买后，在 My Products → Domains 下找到你的域名，点击进入。DNS Management 页面列出了所有记录——A、CNAME、MX、TXT——可以直接在线编辑。界面有些老旧，但能用。

指向服务器：在 @ 处设置 A 记录填入 IP，在 www 处设置 CNAME 指向 @。指向 Vercel 等托管平台：在 @ 处设置 CNAME 填入平台的目标地址。记录类型详情参见 将域名连接到服务器。

第六步：注意营销邮件

6

GoDaddy 会发大量邮件

默认情况下，GoDaddy 会发送营销邮件（"升级您的主机！"、"SSL 即将到期！"、"30 天后续费！"）。续费提醒是有用的，追加销售的那些不是。进入 Account Settings → Communication Preferences，关闭营销邮件，保留交易类邮件。

第七步：何时（以及如何）迁出

7

如果价格让你心疼，就迁走

域名可以在注册商之间转移。大致流程：在原注册商（GoDaddy）解锁域名并申请 EPP/Auth Code；在目标注册商（Porkbun、Cloudflare 等）使用该 Code 发起转移；转移过程需要 5–7 天，同时会为域名续费一年。

转移年的费用按新注册商的价格结算——从 GoDaddy 转到 Cloudflare 的 .com 域名，今后每年可节省约 $10。对于计划持有多年的域名，这笔操作很值得。

注意：注册不足 60 天的域名（新注册或近期转入）无法转移。等满 60 天再操作。

不要轻易把邮箱和注册商捆绑在一起。如果你在 GoDaddy 购买了 Microsoft 365，日后想同时迁移域名和邮箱，就要处理两件事。把邮箱和域名注册分开（Fastmail、Google Workspace 或其他服务），将来迁移成本更低。

从 Porkbun 购买域名

Porkbun 是 Namecheap 本该做到的样子：仪表板简洁、价格透明、免费 WHOIS 隐私保护、免费邮件转发，结账流程没有任何套路。试过它的开发者通常不会回头。唯一的代价是知名度不高。

域名注册商市场分为两个价格层级。第一层——Namecheap、GoDaddy、Squarespace——依靠首年促销价吸引用户，续费时价格上涨，结账时还会推销各种附加服务。第二层——Cloudflare、Porkbun、NameSilo——每年收取统一价格，免费赠送 WHOIS 隐私保护，结账时也不捆绑 SSL 或托管服务。即便只有五个域名，这两层之间的年费差距也相当可观。

Porkbun 在第二层中的定位是"最亲切的那个"。Cloudflare 速度快、价格低，但强制要求使用 Cloudflare 的 DNS。Porkbun 允许你使用任意 DNS 服务商，并且免费提供其他廉价注册商要额外收费的功能（WHOIS 隐私保护、邮件转发、Let's Encrypt SSL、API），界面也没有2008年代的陈旧感。唯一的缺点是品牌知名度不足——如果你告诉一位非技术同事"我在 Porkbun 买了这个域名"，对方可能以为你在开玩笑。

对于为自己项目买域名的开发者来说，这无所谓。本教程将带你完成购买流程，指出我第一次使用时踩过的坑，并解释为什么 Porkbun 特别适合已经在使用 LingCode 的用户。

你将学到什么

  • 为什么 Porkbun 对大多数 TLD 比 Namecheap 更便宜
  • 哪些功能是免费赠送的（WHOIS 隐私保护、邮件转发、SSL）
  • 简洁的单页结账流程
  • 通过 API 从脚本中自动化 DNS 变更
  • 什么情况下你仍然会选择其他注册商

第一步：搜索域名

1

porkbun.com 首页的大搜索框

在首页搜索即可。Porkbun 会同时展示多个 TLD 的可用状态及实际价格——首年和续费价格是同一个数字，没有星号注脚。

你看到的价格就是你实际支付的价格。.com 大约在 $10–11；许多较新的 TLD 在 $5–15 之间。偶尔会有促销活动（尤其是某些 TLD 的首年特价）——那些是真实优惠，不是钓鱼定价，但续费价格才是你从第二年起真正承诺的金额。

第二步：点击"加入购物车"，然后结账

2

不跟你玩套路的结账流程

购物车会显示你的域名及其价格。WHOIS 隐私保护免费且已默认勾选。邮件转发免费且已默认勾选。SSL 免费（通过 Let's Encrypt）且已默认勾选。没有高级 DNS 推销，没有建站工具，没有"立即购买 SSL"的弹窗。

填写联系信息，选择支付方式，点击确认即可。整个流程只有一页，没有"您确定不想添加 X 吗"之类的拦截页面。

"免费邮件转发"的含义：你可以将 hello@yourdomain.com 设置为转发到你现有的邮箱。但仅靠这个，你无法从 hello@yourdomain.com 发送邮件——要实现外发功能，你还需要一个真正的邮箱服务（Fastmail、Google Workspace 等）。对于大多数个人或项目域名来说，免费的收件转发已经够用。

第三步：控制面板——不添乱的 DNS 管理

3

直接在列表中添加记录

购买后点击进入域名。DNS 面板以列表形式展示所有记录，支持直接内联添加和编辑。支持 A、AAAA、CNAME、MX、TXT、SRV 等记录类型，以及 TTL 字段。没有令人困惑的模式切换，也不需要在某个标签页后面找"高级视图"。

对于典型的指向服务器的配置：在 @ 处添加 A 记录填入你的 IP，在 www 处添加 CNAME 指向 @，就完成了。如果对这些记录类型不熟悉，可以参考 DNS 教程。

第四步：用于自动化的 API 访问

4

免费 API 密钥，接口覆盖全面

Porkbun 提供 HTTP API，涵盖你可能想自动化的所有操作：列出域名、获取 DNS 记录、创建/更新/删除记录、管理 URL 转发。API 密钥免费，在账户设置中生成即可。

当你管理多个域名或运行基础设施即代码时，这一点尤为重要。Terraform 的 Porkbun 提供商可用；API 的限速合理；你可以让 Agent（比如运行在 lingcode CLI 中的那个）在脚本中更新 DNS 记录。大型注册商会对 API 访问收费；Porkbun 不会。

第五步：Porkbun 不适合的场景

5

三种真实情况

  • 你的利益相关方在意品牌认知度。如果你在为客户项目购买域名，需要对方看到一个他们熟悉的注册商，那 Porkbun 不合适。知名度更高的选择是有的。
  • 你需要捆绑的邮箱托管服务。Porkbun 的邮件功能仅限转发。如果需要真正的邮箱（让 hello@yourdomain.com 成为可发件地址），你需要单独的邮箱服务商。有些注册商提供捆绑销售；Porkbun 不提供。
  • 你在合同框架下管理大量企业域名。企业级注册商（Mark Monitor、CSC 等）能处理 Porkbun 不支持的法律/合同事宜。个人开发者定价无法满足企业合规需求。

除此之外——你的项目、个人作品集、临时域名、团队工具——Porkbun 都能胜任。

第六步：购买后的操作

6

标准后续步骤

  • 将 DNS 指向你的网站所在位置。参见 将域名连接到服务器。
  • 如果使用 Magic Deploy，将域名提供给托管方并按其 DNS 说明操作——通常是添加一两条 CNAME 记录。
  • 如果你想在自建服务器上启用 SSL，certbot + Let's Encrypt 是标准方案。Porkbun 的免费 SSL 仅适用于 Porkbun 自身的 URL 转发功能，不适用于你自己运行的服务器。
开启账户的双重认证（2FA）。Porkbun 在账户设置中支持 TOTP 2FA。请务必开启。域名账户是攻击者的常见目标；开启 2FA 零成本，防护效果却相当显著。

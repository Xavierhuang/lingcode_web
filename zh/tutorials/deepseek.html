如何在 LingCode 中使用 DeepSeek？

TL;DR：在 Settings → AI Providers → DeepSeek 中填入你的 DeepSeek API key，选择 DeepSeek-V4 Flash（速度优先）或 V4 Pro（质量优先），即可开始对话。CLI 也使用同一个密钥，存储于 macOS Keychain。

快速、实惠、能力出众——将 DeepSeek V4 接入 LingCode，助力快速原型开发与批量构建工作。

为什么选择 DeepSeek？

DeepSeek 的核心优势：

  • 速度：HTML 生成速度最快，设计稿几秒内完成。
  • 成本：比 Claude 便宜 10 倍，非常适合实验和批量工作。
  • 质量：设计任务表现出色。精细程度略逊于 Claude，但能力依然很强。
  • 稳定性：输出一致，极少出现幻觉。
最适合：快速原型、迭代验证、批量构建、学习探索、对成本敏感的工作。

入门

1

创建 DeepSeek 账号

前往 platform.deepseek.com 注册或登录。你需要准备：

  • 电子邮件地址
  • 密码
  • 账单信息（添加支付方式）
2

创建第一个 API key

登录后，在左侧边栏找到 "API Keys" 或 "Keys"，点击 "Create" 或 "+ New"。你会得到一个类似 sk-... 格式的密钥。

请妥善保管此密钥。像对待密码一样对待它，切勿提交到 git 或公开分享。

将 DeepSeek 接入 LingCode

3

打开 playground 并进行配置

前往 /try.html，滚动到页面底部，点击 "Advanced" 展开 provider 配置。

4

粘贴你的 API key

找到 "DeepSeek" 一行，点击输入框并粘贴你的 API key（sk-...）。

点击 "Save"，密钥将安全地存储在你的浏览器中（不会发送到 LingCode 服务器）。

测试连接

5

运行测试提示词

输入一个简单的提示词：

Build a landing page for a coffee shop with a hero, menu preview, and contact form.

点击 "DeepSeek" provider 按钮将其选中，然后点击 "Send"。如果 DeepSeek 在几秒内返回完整的 HTML 页面，说明连接成功！✓

DeepSeek 定价

截至 2026 年，DeepSeek 的价格极为实惠：

  • 典型落地页：约 $0.01–0.05
  • 演示文稿：约 $0.02–0.05
  • 数据看板：约 $0.05–0.10

对比 Claude（每次构建约 $0.10–0.50），DeepSeek 便宜 10 倍。

DeepSeek 模型

DeepSeek 提供两个 V4 版本：

  • V4-Flash：速度快、成本低、质量好，适合大多数任务。
  • V4-Pro：质量略优、速度略慢，适合复杂设计工作。

LingCode 会根据你的提示词自动选择合适的版本，无需手动指定。

实用建议：先试 V4-Flash，速度快且成本低。如果需要更高质量，再针对重要构建切换到 V4-Pro。

何时使用 DeepSeek

  • 快速原型：低成本快速验证想法
  • 批量工作：需要构建 10 个以上的设计？用 DeepSeek 控制成本
  • 学习探索：正在探索 LingCode？DeepSeek 价格亲民
  • 简单设计：落地页、简单看板、快速原型图

何时改用 Claude

  • 高端交付物：客户项目、投资人演示、高要求设计
  • 复杂推理：多步骤或逻辑复杂的设计任务
  • 质量优先于成本时：追求最佳结果，值得多花一点

工作流：DeepSeek + Claude 协作

适合重要项目的高效工作流：

  1. 用 DeepSeek 做原型（快速、低成本）
  2. 方向确定后，用 Claude 重新生成（精细打磨）
  3. 对比两个版本，选出更优的
  4. 导出使用

常见问题

Q：出现"Invalid API key"错误怎么办？

A：请仔细确认是否已从 DeepSeek 控制台完整复制了密钥，注意不要带入多余的空格。

Q：DeepSeek 响应很慢，或提示超出配额？

A：请前往 platform.deepseek.com 查看账号状态。如果触发了速率限制，等待几分钟后重试；如果余额不足，请向账单账户充值。

Q：我现在用的是哪个 DeepSeek 模型？

A：LingCode 会根据任务自动选择 V4-Flash 或 V4-Pro，无需手动指定。

Q：可以同时使用 DeepSeek 和 Claude 吗？

A：可以。分别配置好两个 provider 后，在 playground 中同时选中两者并运行提示词，即可对比结果。

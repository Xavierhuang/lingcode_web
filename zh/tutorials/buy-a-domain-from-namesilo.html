从 NameSilo 购买域名

NameSilo 是一家低调运营近成本定价超过十年的平价域名注册商。界面像 2012 年；价格像成本加一美元；API 可用。对于个人域名组合来说，这是仅次于 Cloudflare 的最便宜的合理选择。

域名注册行业存在两种天然的定价模式。一种是"我们靠追加销售赚钱，域名本身是引流工具"——GoDaddy 和大多数面向消费者的注册商都是这种模式。另一种是"我们靠域名本身的高效运营赚钱，所以价格低廉，不额外收费"——Cloudflare、Porkbun 和 NameSilo 都属于这一类。三者之间的价格差异在绝对数值上并不大；但界面设计的差距却相当悬殊。

NameSilo 的界面是三者中最差的，看起来像 2010 年的注册商——因为它大致就是那时建立的，而且从未大幅改版。他们现有的功能是完整的：支持所有记录类型，免费 WHOIS 隐私保护，免费邮件转发，可用的 API，稳定的运行时间，以及合理的技术支持。界面缺乏美感，但底层注册商服务是可靠的。

对于不在乎控制台外观、很少需要操作域名的开发者（一次配好 DNS、一年不用再管），NameSilo 是合理的选择。而对于追求简洁界面或需要频繁在注册商后台操作的开发者，多花约 1 美元/年选 Porkbun 会是更好的体验。

你将学到

  • NameSilo 的固定定价——每年价格相同
  • 批量注册折扣（适合管理大量域名）
  • 免费包含的内容——WHOIS 隐私保护、邮件转发、DNS
  • API 接口及其使用场景
  • 过时界面带来的利弊权衡

第 1 步：搜索并确认价格

1

显示的价格就是续费价格

在 namesilo.com 搜索域名。显示的价格就是你每年支付的金额——没有首年优惠这种把戏。.com 域名约 9.50 美元，真正接近批发价。

批量折扣会在域名组合规模达到一定数量后生效：在 NameSilo 持有 50 个以上域名时，续费价格每个下降几美分。金额不大，但对于管理大型域名组合的用户（投资者、品牌保护需求等）来说确实实实在在。

第 2 步：结账流程

2

功能完整，但不够美观

加入购物车。购物车页面信息密集——所有选项一次性全部展示，而非逐步引导。WHOIS 隐私保护已默认勾选（免费）；邮件转发已默认勾选（免费）；自动续费默认开启。可选的附加项存在（高级 DNS、某些特定顶级域相关服务），但不会强行推销。

付款确认后，域名立即注册完成。没有动画庆祝页面，只有一条文字确认和你的域名列表。

第 3 步：DNS 管理面板

3

界面过时，但记录类型齐全

点击进入域名，然后选择 DNS 管理。表单极简实用：类型、主机名、值、TTL。填写字段后点击提交即可添加记录；点击记录进行修改后提交即可编辑。

它支持你需要的所有记录类型（A、AAAA、CNAME、MX、TXT、SRV、CAA）。与 Cloudflare 或 Porkbun 相比使用体验略显笨拙，但偶尔修改时也够用。

如果界面让你抓狂：把 DNS 委托出去。将域名的域名服务器改为 Cloudflare 的（即使域名不是从 Cloudflare 购买的，也可以免费使用 Cloudflare DNS），然后在 Cloudflare 的界面管理 DNS。NameSilo 负责注册，Cloudflare 负责解析——两全其美。

第 4 步：API 接口

4

基于 XML 的老式风格，但功能可用

NameSilo 的 API 比 Porkbun 或 Cloudflare 的要老——基于 XML、依赖 GET 参数、文档读起来像是自动生成的。但它能用。现有接口涵盖：列出域名、注册、续费、管理 DNS 记录、管理域名服务器、管理隐私保护。

有社区维护的 Terraform provider。对于自动化场景，API 可以胜任，但不是最顺手的。如果对 API 的开发体验有要求，可以考虑 Porkbun、Cloudflare 或 Gandi——它们都提供基于 JSON 的现代 API。

第 5 步：域名转入与转出

5

标准流程，无阻碍

转入 NameSilo：在当前注册商解锁域名、获取授权码，然后在 NameSilo 发起转入。转入年的费用按 NameSilo 的固定价格计算——从 GoDaddy 或 Namecheap 转过来通常每个域名能省 5 到 10 美元。

转出：同样的流程反向操作。NameSilo 不会刻意阻碍转出——值得特别提一句，因为有些注册商确实会。

第 6 步：什么时候不该选 NameSilo

6

界面是选择其他注册商的主要理由

  • 你需要频繁操作控制台。如果你经常在调整 DNS，日积月累的界面摩擦会让人崩溃。多花几美元选 Porkbun，或者用 Cloudflare。
  • 你需要现代 API 体验。如果一切都用 Terraform 管理，XML API 会拖慢节奏。Cloudflare、Porkbun、Gandi 和 AWS 都提供更好的 API。
  • 你需要强力客户支持。NameSilo 的支持存在且有效，但响应速度不够积极。对于关键域名、需要在一小时内联系到真人的场景，可以考虑付费选 Hover。

第 7 步：购买之后

7

标准步骤

  • 在 NameSilo 管理 DNS，或委托给更好用的 DNS 提供商（Cloudflare DNS 免费）。
  • 在账户设置中开启双因素验证（2FA）。
  • 将域名指向你的网站——参见 DNS 配置指南。
留意续费页面的界面。NameSilo 较旧的界面有时会以不够直观的方式展示附加项（隐私保护、额外年限）。续费时，付款前请仔细核对购物车——看起来只是普通续费，有时可能已经勾选了你没意识到重新确认的额外服务。

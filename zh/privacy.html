隐私政策

我们如何收集、使用和保护你的数据。

翻译说明：本中文版仅为便利读者。如中文与 英文原文 存在歧义或冲突，以英文原文为准。

最后更新：2025 年 12 月 31 日

1. 引言

LingCode（"我们"）致力于保护您的隐私。本隐私政策说明当您使用我们的 macOS 应用 LingCode 时，我们如何收集、使用、披露和保护您的信息。

2. 我们收集的信息

2.1 您提供的信息

  • API Key：当您配置 AI 提供商（OpenAI、Anthropic）时，您可能会提供 API Key。它们被安全地存储在您的 macOS Keychain 中，不会被传输到我们的服务器。
  • 代码与文件：您处理的所有代码与文件都保留在您本地的机器上。我们不会访问、收集或传输您的代码与文件。

2.2 自动收集的信息

  • 使用数据：我们可能会收集匿名使用统计来改进应用（例如功能使用、错误报告）。
  • 崩溃报告：当应用崩溃时，我们可能会收集崩溃日志以协助诊断和修复问题。

3. 我们如何使用您的信息

我们使用收集的信息来：

  • 提供和维护 LingCode 应用
  • 改进并优化应用性能
  • 诊断和修复技术问题
  • 理解用户如何与应用交互

4. 数据存储与安全

4.1 本地存储

您的所有代码、文件和 API Key 都存储在您本地的 Mac 上。我们不维护存放您数据的服务器。

4.2 API Key

API Key 通过 Apple 的安全存储 API 安全地存储在 macOS Keychain 中。我们从不拥有您 API Key 的访问权限。

4.3 AI 提供商通信

当您使用 AI 功能时，您的代码和提示词会直接发送到您配置的 AI 提供商（OpenAI 或 Anthropic）。此通信受这些提供商的隐私政策约束：

  • OpenAI 隐私政策
  • Anthropic 隐私政策

5. 本地专属模式

LingCode 提供"本地专属模式"，确保所有操作留在您本地机器上。启用时，除了 AI API 调用（由您通过 API Key 控制）外，不会有任何数据被传输到外部服务。

6. 第三方服务

LingCode 集成了以下第三方服务：

  • OpenAI：用于 GPT-4 AI 功能（可选，需您的 API Key）
  • Anthropic：用于 Claude AI 功能（可选，需您的 API Key）
  • GitHub：用于 Git 操作（可选，使用您的 Git 凭证）
  • Google Analytics 4：lingcode.dev 网站使用 Google Analytics 统计页面访问量和流量来源。GA 可能会设置 cookie，并接收您的 IP 地址和 User-Agent。我们不会将任何提示词内容、API Key 或工作区文件发送给 Google。您可以使用任何标准的广告/分析拦截器进行屏蔽。

您对这些服务的使用受其各自的隐私政策约束。

7. 数据共享与披露

我们不会向第三方出售、交易或租赁您的个人信息。仅在以下情况下我们可能共享信息：

  • 遵守法律义务
  • 保护和捍卫我们的权利或财产
  • 经您明确同意

8. 您的权利

您有权：

  • 访问和查看您存储的数据（均在您本地 Mac 上）
  • 删除您的数据（卸载应用）
  • 选择退出使用数据追踪（若已实现）
  • 使用本地专属模式以最小化数据传输

9. 儿童隐私

LingCode 不面向 13 岁以下儿童。我们不会故意收集 13 岁以下儿童的个人信息。

10. 本政策的变更

我们可能会不定期更新本隐私政策。我们将通过在此页面发布新的隐私政策并更新"最后更新"日期来通知您任何变更。

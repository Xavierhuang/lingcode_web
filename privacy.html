Privacy Policy

How we collect, use, and protect your data.

Last Updated: December 31, 2025

1. Introduction

LingCode ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our macOS application, LingCode.

2. Information We Collect

2.1 Information You Provide

  • API Keys: When you configure AI providers (OpenAI, Anthropic), you may provide API keys. These are stored securely in your macOS Keychain and are never transmitted to our servers.
  • Code and Files: All code and files you work with remain on your local machine. We do not access, collect, or transmit your code or files.

2.2 Automatically Collected Information

  • Usage Data: We may collect anonymous usage statistics to improve the application (e.g., feature usage, error reports).
  • Crash Reports: If the application crashes, we may collect crash logs to help diagnose and fix issues.

3. How We Use Your Information

We use the information we collect to:

  • Provide and maintain the LingCode application
  • Improve and optimize the application's performance
  • Diagnose and fix technical issues
  • Understand how users interact with the application

4. Data Storage and Security

4.1 Local Storage

All your code, files, and API keys are stored locally on your Mac. We do not maintain servers that store your data.

4.2 API Keys

API keys are stored securely in the macOS Keychain using Apple's secure storage APIs. We never have access to your API keys.

4.3 AI Provider Communication

When you use AI features, your code and prompts are sent directly to the AI provider (OpenAI or Anthropic) you've configured. This communication is subject to the privacy policies of those providers:

  • OpenAI Privacy Policy
  • Anthropic Privacy Policy

5. Local-Only Mode

LingCode offers a "Local-Only Mode" that ensures all operations remain on your local machine. When enabled, no data is transmitted to external services except for AI API calls (which you control via your API keys).

6. Third-Party Services

LingCode integrates with the following third-party services:

  • OpenAI: For GPT-4 AI features (optional, requires your API key)
  • Anthropic: For Claude AI features (optional, requires your API key)
  • GitHub: For Git operations (optional, uses your Git credentials)
  • Google Analytics 4: The lingcode.dev website uses Google Analytics to measure aggregate page views and traffic sources. GA may set cookies and receives your IP address and User-Agent. We do not send any prompt content, API keys, or workspace files to Google. You can opt out with any standard ad/analytics blocker.

Your use of these services is subject to their respective privacy policies.

6a. Sub-processors

For the LingCode Slack integration, we engage the following sub-processors to provide the service. Each processes only the data needed for its function:

  • DigitalOcean — cloud infrastructure hosting for the LingCode backend (servers located in the United States), where workspace install and channel-link data is stored.
  • A large language model provider — used, via the LingCode backend, to generate the bot's replies to messages in which it is mentioned. Message text is sent to generate a reply and is not retained by us.
  • Slack Technologies — the platform the integration runs on, governed by Slack's own terms and privacy policy.

We will update this list as our sub-processors change. Questions may be directed to hhuangweijia@gmail.com.

7. Data Sharing and Disclosure

We do not sell, trade, or rent your personal information to third parties. We may share information only in the following circumstances:

  • To comply with legal obligations
  • To protect and defend our rights or property
  • With your explicit consent

8. Your Rights

You have the right to:

  • Access and review your stored data (all stored locally on your Mac)
  • Delete your data (uninstall the application)
  • Opt out of usage tracking (if implemented)
  • Use Local-Only Mode to minimize data transmission

9. Children's Privacy

LingCode is not intended for children under the age of 13. We do not knowingly collect personal information from children under 13.

10. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last Updated" date.

11. Contact Us

If you have any questions about this Privacy Policy, please contact us:

  • Email: hhuangweijia@gmail.com
  • Phone: (646) 567-1456
  • GitHub Issues: https://github.com/Xavierhuang/LingCode/issues
  • Repository: https://github.com/Xavierhuang/LingCode

12. Open Source

LingCode is an open-source project. You can review the source code at any time to verify our privacy practices. The source code is available at https://github.com/Xavierhuang/LingCode.
